# ING Platform

ING MVP를 위한 Next.js App Router 기반 프론트엔드 프로젝트입니다.

## Stack

- Next.js 16
- TypeScript
- App Router
- ESLint
- CSS Modules
- CSS Variables

## Structure

```text
src/
  app/
  components/
  features/
  lib/
  styles/
```

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Deployment

Vercel에 바로 배포 가능한 기본 구조로 구성되어 있습니다.
